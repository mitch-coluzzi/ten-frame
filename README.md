# Ten-Frame

Browser-based ten-frame emulator — a digital K-2 math manipulative. Tap cells to fill counters, build numbers 0–10. Tablet-first.

Built for Jessie's classroom.

## Stack
- Vanilla HTML/CSS/JS, no build step
- Hosted on Railway (auto-deploy from `main`)

## Local
Open `index.html` in any browser.

## Files
- `index.html` — shell
- `styles.css` — layout, frame, counter styles
- `app.js` — state, render, input
